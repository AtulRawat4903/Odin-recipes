# Odin-recipes
In this project, I am going to build a basic recipe website.
It will consist of a main index page which will have links to few recipes.


After building this Odin-Recipes project, I learned the following beginner web development concepts:

Basic HTML structure (<!DOCTYPE html>, <html>, <head>, <body>)
Headings, paragraphs, lists, and links
Creating multiple webpages
Organizing files and folders properly
Using relative paths like: recipes/pizza.html and ../images/pizza.jpg
Adding images to webpages
Building navigation between pages
Writing semantic content for webpages
