In this project, I built a basic recipe website consisting of a main index page with links to individual recipes.

I applied CSS to enhance the visual design, focusing on page layout, typography, and spacing.

After completing this project, I learned the following beginner web development concepts:

Basic HTML structure: (<!DOCTYPE html>, <html>, <head>, <body>)
Content elements: Headings, paragraphs, lists, and links
CSS styling: Implementing the box model, managing colors, styling fonts, and using pseudo-classes for interactive elements like link hovers
Layout control: Using max-width, margin: 0 auto, and display: block for centering and responsive positioning
Site architecture: Creating multiple webpages and organizing files and folders
Navigation: Using relative paths (e.g., recipes/pizza.html and ../images/pizza.jpg) and building navigation between pages
Media and Semantics: Adding images to webpages and writing semantic content
